About This App

DevOps Notes is a simple note-taking application built using Spring Boot, Maven, and Thymeleaf.

This project was developed to demonstrate a complete CI/CD DevOps pipeline using tools such as Git, Jenkins, Docker, Ansible, Kubernetes, and cloud deployment.

It serves as a practical example for learning and showcasing DevOps automation in real-world projects.
